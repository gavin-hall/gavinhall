# gavinhall.com

This is my personal website.

## Local development

Install dependencies

```bash
npm install
```

Next, create a `.env.local` file in the root of your project and set the `NEXT_PUBLIC_SITE_URL` variable to the site's public URL:

```
NEXT_PUBLIC_SITE_URL=https://gavinhall.com
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.
